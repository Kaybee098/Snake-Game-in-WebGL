# Snake-Game-in-WebGL

Here is a snake game created using WebGL!

## How to Run the Game
* To play this game, you'll need a code editor like Visual Studio Code.
* Install the "Live Server" extension.
* Right-click the `index.html` document and select "Open with Live Server" to launch the game in your preferred web browser.

## How to Play
* The game will start immediately after opening it.
* Use the arrow keys (up, down, left, right) to control the snake.
* Avoid hitting the canvas boundaries (the walls), as this will result in a Game Over.
* Your goal is to eat the blocks of food, which will make your snake longer.
* The game plays a sound in the background every time the snake eats the food.

**Gameplay Notes:**
* Game over if the snake eats itself.
* You cannot oppose your own velocity/motion (e.g., you cannot press 'left' if you are currently moving 'right').
